# Vite Remix

This is a starter template for Remix with Vite with some additional libraries.

You can deploy by connecting Cloudflare Pages and Github without any special setup.

## Features

- Remix with Vite
- Tailwind CSS
- remix-flat-routes
- Deploy to Cloudflare Pages

## Getting Started

```bash
pnpm create remix@latest --template jsparkdev/vite-remix
```

### Cloudflare Pages

- Framework presets : `Remix`
- Build command : `npm run build`
- Build output directory : `build/client`
