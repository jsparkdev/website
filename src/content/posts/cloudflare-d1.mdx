---

아래 명령어를 사용해 데이터베이스를 생성합니다. 예시 코드는 데이터베이ㅅ를 `workers-d1`으로 생성합니다.

```bash
pnpm wrangler d1 create workers-d1
```

콘솔에 출력된 바인딩 정보를 `wrangler.toml` 파일에 추가합니다.

```toml
[[d1_databases]]
binding = "DB" # available in your Worker on env.DB
database_name = "workers-d1"
database_id = "<unique-ID-for-your-database>"
```

다음 명령을 통해 D1 클라이언트 타입을 추가해줍니다.
  
```bash
pnpm wrangler types
```

데이터베이스를 성공적으로 생성했으므로 쿼리를 실행할 수 있습니다. 우선 데이터베이스를 초기화하는데 사용할 스키마 파일을 생성합니다. 이 예시에서는 다음 코드를 `schema.sql` 파일에 저장합니다.

```sql
DROP TABLE IF EXISTS Customers;
CREATE TABLE IF NOT EXISTS Customers (CustomerId INTEGER PRIMARY KEY, CompanyName TEXT, ContactName TEXT);
INSERT INTO Customers (CustomerID, CompanyName, ContactName) VALUES (1, 'Alfreds Futterkiste', 'Maria Anders'), (4, 'Around the Horn', 'Thomas Hardy'), (11, 'Bs Beverages', 'Victoria Ashworth'), (13, 'Bs Beverages', 'Random Name');
```

wrangler는 execute 명령을 사용하여 쿼리를 실행할 수 있습니다. 로컬 데이터베이스에 쿼리를 적용하기 위해 다음 명령을 실행합니다.

```bash
pnpm wrangler d1 execute workers-d1 --local --file=./schema.sql
```

로컬 데이터베이스가 제대로 초기화되었는지 확인하기 위해 테이블을 조회합니다.

```bash
pnpm wrangler d1 execute workers-d1 --local --command="SELECT * FROM Customers"
```

이렇게 로컬 데이터베이스를 초기화하는데 성공하였습니다. 이제 Workers 코드에서 데이터베이스를 사용해야 합니다. 메인 파일의 내용을 다음과 같이 수정합니다.

Workers에서는 D1 데이터베이스와 상호작용하기 위한 D1 클라이언트 API를 제공합니다.

```ts
export default {
	async fetch(request: Request, env: Env) {
		const { pathname } = new URL(request.url);

		if (pathname === "/api/beverages") {
			const { results } = await env.DB.prepare(
				"SELECT * FROM Customers WHERE CompanyName = ?",
			)
				.bind("Bs Beverages")
				.all();

			return Response.json(results);
		}

		return new Response(
			"Call /api/beverages to see everyone who works at Bs Beverages",
		);
	},
};
```

`prepare` 메서드는 쿼리를 준비하고, `bind` 메서드는 쿼리에 바인딩할 값을 설정합니다. 

`prepare` 메서드가 반환한 값의 `all` 메서드는 모든 행을 객체 배열로 반환합니다.

로컬 개발 모드에서 이 작업을 수행했다면, 이제 원격 데이터베이스에도 이 작업을 적용합니다.

```bash
pnpm wrangler d1 execute workers-d1 --remote --file=./schema.sql
pnpm wrangler d1 execute workers-d1 --remote --command="SELECT * FROM Customers"
pnpm wrangler deploy
```
