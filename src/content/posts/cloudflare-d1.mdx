D1 데이터베이스는 Cloudflare Pages에 호스팅된 프로젝트 또는 Cloudflare Workers에서 사용할 수 있는 서버리스 SQLite 기반 데이터베이스입니다.

## 데이터베이스 생성

D1 데이터베이스는 wrangler를 사용하여 생성할 수 있습니다. 다음 명령을 통해 wrangler에 로그인 후, 데이터베이스를 생성합니다.

```bash
pnpm wrangler login
pnpm wrangler d1 create my-database
```

## 데이터베이스 바인딩

생성한 데이터베이스를 Workers와 연결하기 위해 바인딩을 설정해야 합니다. 데이터베이스 생성 후 콘솔에 출력된 아래 메시지를 `wrangler.toml` 파일에 추가합니다.

```toml
[[d1_databases]]
binding = "DB" # available in your Worker on env.DB
database_name = "my-database"
database_id = "<unique-ID-for-your-database>"
```

## 쿼리 실행

wrangler를 사용하여 데이터베이스에 쿼리를 실행할 수 있습니다. 쿼리는 SQL파일로 제공하거나 직접 명령을 전달할 수 있습니다.

### SQL 파일로 쿼리 실행

`schema.sql` 파일을 생성한 후, 다음 코드를 추가합니다.

이 코드는 `Customers` 테이블을 생성하고, 몇 가지 데이터를 삽입합니다.

```sql
DROP TABLE IF EXISTS Customers;
CREATE TABLE IF NOT EXISTS Customers (CustomerId INTEGER PRIMARY KEY, CompanyName TEXT, ContactName TEXT);
INSERT INTO Customers (CustomerID, CompanyName, ContactName) VALUES (1, 'Alfreds Futterkiste', 'Maria Anders'), (4, 'Around the Horn', 'Thomas Hardy'), (11, 'Bs Beverages', 'Victoria Ashworth'), (13, 'Bs Beverages', 'Random Name');
```

다음 명령을 사용하여 로컬 데이터베이스에 `schema.sql`파일을 실행할 수 있습니다.

```bash
pnpm wrangler d1 execute my-database --local --file=./schema.sql
```

### 직접 쿼리 실행

초기화된 로컬 데이터베이스에 직접 쿼리를 실행합니다.

이 명령은 `Customers` 테이블의 모든 데이터를 반환합니다.

```bash
pnpm wrangler d1 execute my-database --local --command="SELECT * FROM Customers"
```

## Workers에서 쿼리 실행

Workers에서 코드를 통해 쿼리를 실행하기 위해 D1 클라이언트를 사용합니다.

우선, D1 클라이언트에 자동완성을 제공하기 위해 다음 명령을 사용하여 타입을 정의합니다.

```bash
pnpm wrangler types
```

Workers fetch 핸들러의 `env` 매개변수를 통해 D1 클라이언트를 사용할 수 있습니다.

D1 클라이언트는 `prepare` 메서드를 사용하여 쿼리를 실행하고 결과를 반환합니다.

```ts
export default {
	async fetch(request: Request, env: Env) {
		const { results } = await env.DB.prepare(
			"SELECT * FROM Customers WHERE CompanyName = ?",
		)
			.bind("Bs Beverages")
			.all();

		return Response.json(results);
	},
};
```

## 데이터베이스 배포

개발 모드에서의 작업이 모두 완료되었다면, 다음 명령을 실행하여 해당 작업을 원격 데이터베이스에 적용하고 Workers에 배포합니다.

```bash
pnpm wrangler d1 execute my-database --remote --file=./schema.sql
pnpm wrangler d1 execute my-database --remote --command="SELECT * FROM Customers"
pnpm wrangler deploy
```
