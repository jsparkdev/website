`animation-composition` 속성은 CSS 속성으로 지정한 기본값과 `@keyframes` 규칙에서 설정한 효과값을 결합하는 방법을 설정합니다.

이 속성은 다음 세 가지 값을 사용할 수 있습니다.

- [replace](#replace)
- [add](#add)
- [accumulate](#accumulate)

## replace

`replace`는 기본값을 무시하고 `@keyframes` 규칙에서 설정한 효과값만 사용합니다. 이 값은 별도로 `animation-composition` 속성을 지정하지 않아도 기본값으로 사용됩니다.

```css
.box {
  transform: translateX(100px) rotate(45deg);
  background-color: green;
  animation: slide 4s linear infinite;
}

@keyframes slide {
  40% {
    transform: translateX(100px) rotate(45deg);
    background-color: hotpink;
  }
  100% {
    transform: translateX(200px) rotate(45deg);
    background-color: red;
  }
}
```

시작하는 효과값이 존재하지 않으므로 기존값을 사용합니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;
```

40% 지점에서는 효과값이 존재하므로 기존 값은 무시되고, 효과값이 사용됩니다.

시작 지점의 값과 40% 지점의 값의 차이는 오직 색상 뿐이므로 동일한 위치에서 색상만 변경됩니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(100px) rotate(45deg);
background-color: hotpink;
```

100% 지점에서도 효과값이 존재하므로 기존 값은 무시되고, 효과값이 사용됩니다.

40% 지점의 값과 100% 지점의 값의 차이로 인해 요소는 오른쪽으로 100px (기존 지점에서 오른쪽으로 200px 이동) 이동합니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(200px) rotate(45deg);
background-color: red;
```

## add

`add`는 기본값과 `@keyframes` 규칙에서 설정한 효과값을 더합니다.

```css
.box {
  transform: translateX(100px) rotate(45deg);
  background-color: green;
  animation: slide 4s linear infinite;
  animation-composition: add;
}

@keyframes slide {
  40% {
    transform: translateX(100px) rotate(45deg);
    background-color: hotpink;
  }
  100% {
    transform: translateX(200px) rotate(45deg);
    background-color: red;
  }
}
```

시작하는 효과값이 존재하지 않으므로 기존값을 사용합니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;
```

40% 지점에서는 효과값이 존재하므로 기존 값에 효과값이 추가됩니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(100px) rotate(45deg);
background-color: hotpink;

# 기존값과 효과값의 합
transform: translateX(100px) rotate(45deg) translateX(100px) rotate(45deg);
background-color: green + hotpink; # rgb(0, 128, 0) + rgb(255, 105, 180) = rgb(255, 233, 180)
```

100% 지점에서도 효과값이 존재하므로 기존 값에 효과값이 추가됩니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(200px) rotate(45deg);
background-color: red;

# 기존값과 효과값의 합
transform: translateX(100px) rotate(45deg) translateX(200px) rotate(45deg);
background-color: green + red; # rgb(0, 128, 0) + rgb(255, 0, 0) = rgb(255, 128, 0)
```

## accumulate

`accumulate`는 기본값과 `@keyframes` 규칙에서 설정한 효과값을 누적합니다.

```css
.box {
  transform: translateX(100px) rotate(45deg);
  background-color: green;
  animation: slide 4s linear infinite;
  animation-composition: accumulate;
}

@keyframes slide {
  40% {
    transform: translateX(100px) rotate(45deg);
    background-color: hotpink;
  }
  100% {
    transform: translateX(200px) rotate(45deg);
    background-color: red;
  }
}
```

시작하는 효과값이 존재하지 않으므로 기존값을 사용합니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;
```

40% 지점에서는 효과값이 존재하므로 기존 값에 효과값이 누적됩니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(100px) rotate(45deg);
background-color: hotpink;

# 기존값과 효과값의 누적
transform: translateX(200px) rotate(90deg); # translateX(100px +_ 100px) rotate(45deg + 45deg)
background-color: green + hotpink; # rgb(0, 128, 0) + rgb(255, 105, 180) = rgb(255, 233, 180)
```

100% 지점에서도 효과값이 존재하므로 기존 값에 효과값이 누적됩니다.

```css
# 기존값
transform: translateX(100px) rotate(45deg);
background-color: green;

# 효과값
transform: translateX(200px) rotate(45deg);
background-color: red;

# 기존값과 효과값의 누적
transform: translateX(300px) rotate(90deg); # translateX(100px + 200px) rotate(45deg + 45deg)
background-color: green + red; # rgb(0, 128, 0) + rgb(255, 0, 0) = rgb(255, 128, 0)
```
